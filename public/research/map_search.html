NOTES: You do not need to fill in every blank. You only need to type part of a name or word for the search engine to find results. Typing part of a word will return all of the derivatives; searching for the character “A” will return any word with the letter “A” in it.

HINT: If you want to find all the names that begin with the letter B, type ^B in the Name blank.

Advanced Search Tips

+ after a number or letter will return all occurrences of records with one or more of the characters preceding the +. Thus Cab+ finds Cab, Cabby, and so on.

* after a number or letter will return all occurrences of records with zero or more of the characters preceding the *. Thus ab* finds sad, cab, sabbath, and so on.

? after a number or letter indicates that the number or letter preceding
the ? is optional. Thus ou?r finds color, colour, and so on.

. (the period) puts any character or number in its place that is found in the records searched. Thus n.t finds not, nit, nat, and so on.
